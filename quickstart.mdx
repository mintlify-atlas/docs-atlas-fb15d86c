## Prerequisites

You need R 4.0 or later and the following CRAN packages. Install them all with one command:

```r
install.packages(c("dplyr", "MASS", "neuralnet", "caret", "ggplot2"))
```

<Note>
  `pred_sumo.R` only requires `dplyr`, `MASS`, and `neuralnet`. `caret` and `ggplot2` are additional dependencies for the full evaluation workflow in `sumo.Rmd`.
</Note>

## Get the code

```bash
git clone https://github.com/SamReeves/sumo.git
cd sumo
```

The repository contains three files you will work with:

| File | Purpose |
|------|---------|
| `sumo.csv` | Training data and match input |
| `pred_sumo.R` | Prediction script (GLM + neural network) |
| `sumo.Rmd` | Full evaluation and report |

## Run your first prediction

<Steps>
  <Step title="Inspect the data">
    Open `sumo.csv` to see the format. Each row is one match. The nine columns are `weight1`, `weight2`, `wins1`, `wins2`, `age1`, `age2`, `height1`, `height2`, and `result`.

    ```csv sumo.csv
    weight1,weight2,wins1,wins2,age1,age2,height1,height2,result
    401,379,6,5,35,31,190,184,1
    344,370,8,11,36,34,189,192,1
    351,245,3,2,22,27,189,169,0
    335,291,3,12,32,30,177,176,0
    ```

    Rows with `result = 1` or `result = 0` are historical matches used for training. A blank `result` field marks the match you want to predict.
  </Step>

  <Step title="Add an undecided match">
    Append a row to `sumo.csv` with the two wrestlers' stats and leave `result` blank. The trailing comma is required — it produces an `NA` when R reads the file.

    ```
    385,362,9,7,28,33,188,181,
    ```

    Columns in order: `weight1`, `weight2`, `wins1`, `wins2`, `age1`, `age2`, `height1`, `height2`, `result` (blank).

    <Warning>
      Only one undecided row at a time is supported. The script uses a scalar `if` comparison that will fail if `undecided` contains more than one row.
    </Warning>
  </Step>

  <Step title="Source the prediction script">
    Open an R console in the `sumo/` directory and run:

    ```r
    source("pred_sumo.R")
    ```

    In RStudio, open `pred_sumo.R` and press `Ctrl+Shift+S` (or `Cmd+Shift+S` on macOS) to source it.

    <Note>
      The script must be run from the same working directory as `sumo.csv`, or update the path inside `sumo.read('sumo.csv')`.
    </Note>
  </Step>

  <Step title="Read the predictions">
    Both models print their prediction to the console, followed by the raw numeric value.

    ```
    [1] "GLM says left."
            1
     1.382941
    [1] "NN says left."
              [,1]
    [1,] 0.6173284
    ```

    - **GLM** (`bin.ans`): a log-odds value. Positive → left wrestler wins; negative → right wrestler wins.
    - **NN** (`nn.ans`): a probability. `>= 0.5` → left wrestler wins; `< 0.5` → right wrestler wins.

    <Tip>
      When the two models disagree, trust the GLM — it achieves approximately 70% accuracy on held-out data, consistently outperforming the neural network on this dataset.
    </Tip>
  </Step>
</Steps>
